# Custom Assets for AdvantageScope

This repository contains a custom assets for use with **AdvantageScope** for FRC.  
By cloning or downloading this repository and transferring's it contents into AdvantageScope, you can load a custom TurretBot for yaw-visualization purposes as well as a FRC 2026 Rebuilt Fuel game piece.

### Notes
- AdvantageScope automatically detects assets based on folder names: renaming a file may alter the file's portrayed purpose.

## Asset Types & Formats (in this repository)

### **Robots**
- **3D model:** `.glb`
- **Robot config:** `.json` defining joints, camera positions, and transforms
